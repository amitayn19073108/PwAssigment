
HTML Entities:
HTML entities are special character codes used to representcharacters that have a specific meaning in HTML, or characters that are not easily typed using a standard keyboard. They are used to ensure that these characters are displayed correctly in web pages, as well as to prevent the characters from being interpreted as part of the HTML code. HTML entities are composed of an ampersand (&), followed by a code or name, and ending with a semicolon (;). 
Here are some common HTML entities:

5 Commonly Used HTML Entities:

&lt; - Represents the less-than symbol <.

Example: <p>This is an example &lt;em&gt;text&lt;/em&gt;.</p>


&gt; - Represents the greater-than symbol >.

Example: <p>For more information, visit our website &gt; <a href="#">here</a>.</p>


&amp; - Represents the ampersand symbol &.

Example: <p>For further details, contact us &amp; we'll assist you.</p>


&quot; - Represents a double quotation mark "

Example: <p>The title of the book is "The Great Gatsby".</p>


&copy; - Represents the copyright symbol Â©.

Example: <p>&copy; 2023 ACME Corporation. All rights reserved.</p>
